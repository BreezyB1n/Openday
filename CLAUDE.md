
# Openday
Openday是海外留学生申请的开放通知平台。用户订阅目标项目偏好，申请窗口开放时第一时间收到邮件通知。面对计划申请海外研究生/本科的中国学生，有明确目标院校但申请经验有限。

## 技术栈

- **框架**: Next.js 16 (App Router)
- **语言**: TypeScript
- **样式**: Tailwind CSS v4
- **包管理**: pnpm
- **运行时**: React 19

## 常用命令

```bash
pnpm dev      # 启动开发服务器
pnpm build    # 构建生产版本
pnpm lint     # 代码检查
```

## 项目结构

```
app/          # Next.js App Router 页面
prd/          # 产品需求文档
public/       # 静态资源
```

## P0 核心模块（MVP）
1. **用户注册/登录** - 邮箱+密码，7天登录态
2. **偏好设置** - 学位层级、专业方向（≤5个）、目标国家（≤3个）
3. **项目数据库** - 200个热门项目（QS Top 200），含申请要求全字段
4. **申请开放通知** - 邮件推送，偏好3维度AND匹配，1小时内完成
5. **项目详情页** - 截止日期倒计时、语言/GPA要求、申请官链
